# Tranzio Fitness Website

A premium, Awwwards-quality marketing website for the **Tranzio** ecosystem. The site is built with React 19, Vite, Tailwind CSS, GSAP, and Framer Motion. It includes interactive dashboard widget visualizations instead of generic phone/device mockups.

## Features

- **Hero Dashboard Banner Card:** A floating 3D dashboard visualization grid comprising the interactive modules:
  - **Weight Tracker Chart:** An animated SVG line graph mapping weight trends.
  - **Workout Summary:** Concentric progress rings illustrating workout splits and duration.
  - **Calorie Tracker:** Active macros breakdowns and water intake tracker.
  - **Today's Goal Tracker:** Interactive daily streaks and step counter.
- **Scroll Storytelling features:** Pinned layout segments driven by GSAP ScrollTrigger to morph app modules as you scroll.
- **Deficit Forecasting:** Interactive metabolic calculator projecting weekly weight changes based on intake and expenditure.
- **App Download Integrations:** Custom visual links for iOS App Store and Google Play downloads.
- **Modular Components & Responsive Design:** Optimized for Desktop, Tablet, and Mobile displays.

## Technology Stack

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS (v3) + PostCSS
- **Smooth Scroll:** Lenis
- **Animations:** GSAP, GSAP ScrollTrigger, Framer Motion
- **Icons:** React Icons

## Getting Started

### Installation

1. Install all dependencies:
   ```bash
   npm install
   ```

2. Run the local development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
